# 照片动画功能说明

## 🎬 新增动画效果

当访客（非创建者）打开红包时，照片会经历以下动画序列：

### 动画流程

1. **初始显示（0-3秒）**
   - 照片从红包中弹出
   - 停留在红包上方
   - 保持静止状态 3 秒

2. **放大阶段（3-5秒）**
   - 照片开始放大
   - 逐渐放大到 5 倍大小
   - 持续 2 秒
   - 几乎铺满整个屏幕

3. **漂浮阶段（5秒后）**
   - 照片缩小到正常大小
   - 开始在屏幕内随机移动
   - 碰到边界会反弹
   - 同时进行无规则旋转
   - 持续漂浮直到关闭页面

---

## 🎯 技术实现

### 动画状态

使用 4 个状态控制动画：
- `initial` - 初始弹出
- `pause` - 停留 3 秒
- `expand` - 放大到铺满屏幕
- `floating` - 漂浮和旋转

### 物理效果

**碰撞检测**：
- 实时检测照片位置
- 碰到边界时反弹
- 改变运动方向

**随机运动**：
- 随机初始速度（2-6 像素/帧）
- 随机旋转速度
- 碰撞时改变旋转方向

**边界约束**：
- 确保照片不会超出屏幕
- 自动调整位置保持在可见范围

## 👀 用户体验

### 访客模式
1. 点击红包上的"開"按钮
2. 照片弹出并停留 3 秒（可以欣赏照片）
3. 照片放大铺满屏幕（2 秒）
4. 照片缩小并开始漂浮
5. 照片在屏幕内随机移动和旋转
6. 背景音乐同时播放

### 创建者模式
- 创建者打开红包时**不会触发漂浮动画**
- 照片保持静态显示
- 方便创建者预览和管理内容

## ⚙️ 动画参数

可以调整的参数（在代码中）：

```typescript
// 停留时间
setTimeout(() => setAnimationStage('pause'), 100);
setTimeout(() => setAnimationStage('expand'), 3100); // 3秒后放大

// 放大倍数
animate={{ scale: 5 }} // 放大到5倍

// 移动速度
let vx = Math.random() * 4 + 2; // 2-6 像素/帧
let vy = Math.random() * 4 + 2;

// 旋转速度
let rotationSpeed = (Math.random() - 0.5) * 3; // -1.5 到 1.5 度/帧

// 照片大小（漂浮时）
className="w-48 h-60" // 宽48 高60
```

## 🐛 注意事项

### 性能优化
- 使用 `requestAnimationFrame` 实现流畅动画
- 组件卸载时自动清理动画
- 避免内存泄漏

### 响应式设计
- 自动适配不同屏幕尺寸
- 边界检测基于窗口大小
- 移动端和桌面端都能正常工作

### 浏览器兼容性
- 使用 Motion (Framer Motion) 库
- 支持现代浏览器
- 自动降级处理

## 🔧 自定义建议

### 调整停留时间
修改第一个 setTimeout 的延迟：
```typescript
setTimeout(() => {
  setAnimationStage('expand');
}, 3100); // 改为 5100 可以停留 5 秒
```

### 调整移动速度
修改速度范围：
```typescript
let vx = Math.random() * 6 + 3; // 更快：3-9
let vy = Math.random() * 6 + 3;
```

### 调整旋转速度
修改旋转速度：
```typescript
let rotationSpeed = (Math.random() - 0.5) * 5; // 更快旋转
```

### 调整放大倍数
修改 scale 值：
```typescript
animate={{ scale: 3 }} // 放大到3倍（更小）
animate={{ scale: 8 }} // 放大到8倍（更大）
```
