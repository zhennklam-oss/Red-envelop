# 红包惊喜应用 - 使用说明

## 📦 功能介绍

这是一个节日主题的红包应用，允许您上传照片，然后通过分享链接让局域网内的其他人打开"红包"查看惊喜照片。

---

## 🚀 快速开始

### 1. 安装依赖

首先确保您已安装 Node.js (建议 v18 或更高版本)，然后在项目目录下运行：

```bash
npm install
```

### 2. 启动服务器

```bash
npm run dev
```

启动成功后，您会看到类似以下信息：

```
========================================
🎉 红包惊喜服务器已启动！
========================================
本地访问: http://localhost:3000
局域网访问: http://192.168.1.100:3000

创建者链接: http://192.168.1.100:3000?key=creator_secret_123
分享链接: http://192.168.1.100:3000
========================================
```

---

## 📝 使用步骤

### 作为创建者（上传照片）

1. **打开创建者链接**
   - 在浏览器中访问带有 `?key=creator_secret_123` 参数的链接
   - 例如：`http://192.168.1.100:3000?key=creator_secret_123`

2. **上传照片**
   - 点击"选择并上传照片"按钮
   - 选择一张照片（支持常见图片格式）
   - 等待上传完成

3. **分享链接**
   - 上传成功后，点击"复制分享链接"按钮
   - 将链接发送给同一局域网内的朋友

### 作为访客（查看惊喜）

1. **打开分享链接**
   - 在浏览器中访问朋友发送的链接
   - 例如：`http://192.168.1.100:3000`

2. **打开红包**
   - 看到红包界面后，点击中间的"開"按钮
   - 红包会打开，显示惊喜照片

---

## 🌐 网络要求

- **创建者和访客必须在同一局域网内**（例如连接同一个 WiFi）
- 确保防火墙允许 3000 端口的访问
- 如果其他设备无法访问，请检查：
  - 是否连接同一网络
  - Windows 防火墙设置
  - 路由器是否启用了 AP 隔离

---

## 🔧 常见问题

### Q: 其他设备无法访问怎么办？

**A:** 检查 Windows 防火墙设置：

1. 打开"Windows Defender 防火墙"
2. 点击"允许应用通过防火墙"
3. 找到 Node.js 或允许端口 3000
4. 确保"专用"和"公用"都已勾选

或者临时关闭防火墙测试（不推荐长期使用）。

### Q: 如何更换照片？

**A:** 使用创建者链接访问，点击"更换照片"按钮即可删除当前照片并上传新的。

### Q: 照片存储在哪里？

**A:** 照片以 Base64 格式存储在项目目录下的 `surprise.db` SQLite 数据库中。

### Q: 如何修改管理员密钥？

**A:**
1. 修改 `src/App.tsx` 第 10 行的 `ADMIN_KEY` 常量
2. 修改 `server.ts` 第 34 行的密钥验证
3. 重启服务器

---

## ⚠️ 注意事项

1. **安全性**：当前密钥硬编码在代码中，仅适合局域网内部使用
2. **单一照片**：每次只能存储一张照片，更换会覆盖旧照片
3. **文件大小**：建议上传的照片不超过 5MB，以确保加载速度
4. **浏览器兼容性**：建议使用 Chrome、Edge、Firefox 等现代浏览器
